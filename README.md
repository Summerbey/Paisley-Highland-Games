## Project Overview
A full-stack web application for managing Highland Games events and competitor registrations.

## Technologies Used
**Backend:**
- Node.js & Express.js
- MongoDB with Mongoose
- RESTful API

**Frontend:**
- React 18 with TypeScript
- Vite build tool
- React Router
- Axios for API calls
