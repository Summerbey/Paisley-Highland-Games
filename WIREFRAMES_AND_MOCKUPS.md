# PAISLEY HIGHLAND GAMES - WIREFRAMES & UI MOCKUPS

## Design System

### Color Palette
- **Primary Blue**: #1E3A8A (Deep blue for headers and primary actions)
- **Accent Gold**: #F59E0B (Warm gold for CTAs and highlights)
- **Success Green**: #10B981 (Forest green for success states)
- **Heritage Purple**: #8B5CF6 (Heather purple for cultural elements)
- **Neutral Grays**: #F3F4F6 (light) to #1F2937 (dark)

### Typography
- **Headers**: Inter Bold, 24-32px
- **Body**: Inter Regular, 16px
- **Small Text**: Inter Regular, 14px
- **Line Height**: 1.5

### Spacing System
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px

### Breakpoints
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## Design Principles Applied

### 1. **Progressive Disclosure**
- Information revealed in layers (summary â†’ detail â†’ actions)
- Multi-step forms reduce cognitive load
- Expandable sections for advanced options

### 2. **Visual Hierarchy**
- Large hero images draw attention
- Bold typography for headers
- Color-coded status badges
- Icons support text labels

### 3. **Accessibility**
- High contrast ratios (WCAG AA compliant)
- Clear focus states
- Screen reader friendly labels
- Keyboard navigation support

### 4. **Mobile Responsiveness**
- Touch-friendly button sizes (minimum 44x44px)
- Collapsible navigation
- Stacked layouts on mobile
- Optimized image sizes

### 5. **Feedback & States**
- Loading states for async operations
- Success/error messages
- Progress indicators
- Disabled state styling

### 6. **Consistency**
- Repeated patterns across pages
- Consistent button styles
- Uniform spacing
- Predictable navigation

---

## Interactive Prototype Features

### Micro-interactions
- Button hover effects (color shift, shadow)
- Form input focus (border highlight)
- Card hover (subtle lift, shadow)
- Loading spinners
- Success checkmarks
- Error shake animations

### Transitions
- Page transitions: Fade in (300ms)
- Modal open/close: Scale + fade (200ms)
- Accordion expand: Height transition (250ms)
- Tab switching: Fade (150ms)

### Responsive Behaviors
- **Desktop**: Three-column layouts, hover states
- **Tablet**: Two-column layouts, touch optimization
- **Mobile**: Single column, hamburger menu, bottom navigation
